-> Configura o pc com nome e e-mail do responsável pelo código
git config --global user.name "Gabriel Modenese Faggian Abila"
git config --global user.email gabrielmfa2@gmail.com

-> Navegando no cmd
cd nome-da-pasta // entra em uma pasta
cd..             // volta uma pasta
cd /             // vai para o C:

-> Inicializa um repositório git (utilizado apenas 1 vez)
git init

-> Verificar o status do repositório
git status

-> Adicionar os novos arquivos no git para versionar
git add .

-> Salva uma versão dos arquivos
git commit -m "Por que estou salvando???"

-> Envia informação do computador para o github
git push

-> Baixa a atualização dos arquivos do Github para uma pasta já existente
git pull

-> Sincroniza sem alterar a pasta local verificando mudanças no Github
git fetch
